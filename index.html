AI coding tools comparison

Google Antigravity、Claude Code、Codex：
主要差別與免費帳號實用比較

這一頁不是比「誰最強」，而是比「定位差在哪裡、免費帳號能怎麼用、以及對一般教學 / 示範 / 個人開發哪一種更實用」。

整理重點：官方產品定位、免費方案可用性、免費使用者的優缺點與實務選擇。
一句話

Antigravity

比較像agent-first AI IDE：多代理、工作區、瀏覽器操作、圖形化體驗較完整。

一句話

Claude Code

比較像深度 codebase 代理工具：很強，但對純免費帳號不算最友善。

一句話

Codex

比較像最容易免費上手的 coding agent：官方明列 ChatGPT Free 可用，但偏短任務。

逐一說明

每個工具到底像什麼？

Google Antigravity
最像「代理人導向的 AI 開發環境」
官方把它描述成 agent-first / AI IDE，重點在 Agent Manager、多工作區、多代理、編輯器、Browser Use、Artifacts 與 Knowledge。
主要特色
  • 偏圖形化操作，不只是命令列
  • 強調多代理平行工作
  • 可結合 browser / workspace 工作流
  • 對示範教學較直觀
你可以怎麼理解

如果 Claude Code / Codex 比較像「用 agent 幫你寫程式」，Antigravity 更像「一個把 agent 當主角的開發桌面」。

Claude Code
最像「能深入大型 codebase 的工程代理」
官方定位是 agentic coding tool：可讀 codebase、編輯檔案、跑命令、整合開發工具，適合真正進入專案做事。
主要特色
  • 理解大型專案能力強
  • 適合重構、除錯、跨檔案修改
  • 偏工程工作流，而不是展示型 UI
  • terminal / IDE / web 都可串接
你可以怎麼理解

如果你已經有一個真實專案，要找一個「很像資深工程夥伴」的代理，Claude Code 的定位最清楚。

Codex
最像「最容易先免費試用的 coding agent」
OpenAI 官方把 Codex 定位為 software development 的 coding agent，可協助寫程式、理解陌生程式庫、做 code review、debug、執行重複性開發任務。
主要特色
  • 官方明確支援 ChatGPT Free
  • 很適合快速產生、修改、解釋程式
  • 適合短任務、快速試跑
  • 新手試用門檻最低
你可以怎麼理解

如果你想先免費試試「AI 幫我做 coding task 到底長怎樣」，Codex 是三者裡最容易立刻上手的。

最像什麼 「AI IDE + 多代理桌面」 「工程用重型代理」 「快速可用的 coding assistant / agent」
實用判斷

免費帳號的優缺點：講白話一點

Google Antigravity

優點 官方就有免費方案、UI 直觀、多代理概念清楚，對教學展示、工作流示範很有吸引力。

缺點 免費額度採週配額，但官方沒有把精確數字講很清楚；如果用得重，會比較難預估何時撞限。

Claude Code

優點 對真實工程專案、長程式碼上下文、codebase 理解非常有代表性；定位成熟。

缺點 以「免費帳號」角度來看最不友善。不是不能碰，但通常不會是零成本、零門檻直接開始。

Codex

優點 官方直接說 Free 可用，最適合先體驗 coding agent。對小任務、課堂 demo、快速測試很方便。

缺點 免費版官方用語就是 quick coding tasks，意思很明白：長時間、高頻、重任務不宜期待太多。

選擇建議

如果是不同使用情境，我會怎麼選？

情境 比較建議 原因
想先免費試試看 AI coding agent Codex 官方明列 Free 可用，最適合快速上手
想做圖形化示範、強調多代理與流程展示 Antigravity 更像完整 AI IDE / 多代理工作台
要處理較真實的專案、跨多檔案修改與重構 Claude Code 工程代理定位最強，但免費帳號不是優勢
要做課堂 demo、短時間展示 Codex 或 Antigravity 一個免費直上手，一個畫面與 workflow 更好講
完全不想先付費 先試 Codex，再看 Antigravity 這兩個對免費使用者的官方訊息最明確
注意事項

有兩個地方要特別說明

  • Google Antigravity 的名稱辨識：這裡採用目前官方站點 antigravity.google 所對應的產品來解釋；不是網路迷因版的「Google Gravity」。
  • 免費可用 ≠ 無限制：Antigravity 與 Codex 雖然都有官方明列免費可用，但都不是「無上限重度使用」；只是它們對免費用戶的路徑比 Claude Code 清楚得多。
