Overview

Reimagining physics lessons with LLM-driven agents

OpenMAIC Physics is a local teaching prototype inspired by the OpenMAIC research concept, adapted for Traditional Chinese high-school physics. It is not an official OpenMAIC branch; it borrows the same classroom metaphor and generation pipeline for teaching design.

教師輸入主題、年級、時間與學生迷思後，系統先產出課程大綱，再把每個大綱節點轉成 slides、quizzes、simulations、PBL activities、whiteboard scripts 與 handouts。安全與模型呼叫細節則集中在 Architecture 與 FAQ。

Platform Highlights

  • Multi-agent classroom：AI teacher、TA、AI classmates、physics checker
  • Rich scene types：slides、quizzes、simulations、PBL、whiteboard、handout
  • Two-stage pipeline：topic/document → structured outline → scene content
  • Teacher-ready outputs：course cards、misconception cards、assessment cards
  • 安全部署：公開頁面不存 API key；OAuth 只在 Hermes 代理端
  • 適合手機開啟、共備展示、課堂投影與講義匯出
Features

A complete AI classroom experience

以下功能刻意對齊 OpenMAIC 的主要 scene vocabulary，但轉成繁中高中物理老師能直接使用的共備與課堂產出。

⌁

Intelligent Slide Lectures

把力矩、等電位線、碰撞、光電效應等主題拆成概念開場、圖像推導、公式收束與檢核提問。

✓

Interactive Quizzes & AI Grading

產生概念題、診斷題、遷移題與評分規準，協助老師辨認學生錯因。

◎

Interactive HTML Simulations

用簡潔 SVG / HTML 呈現物理量關係，例如力臂、向量方向、等位線與能量圖像。

▦

Project-Based Learning

把單一概念延伸成小組任務：預測、操作、解釋、反駁與共同產出。

✎

Collaborative Whiteboard

提供逐格板書腳本：何時只畫情境、何時補方向、何時寫公式、何時做反例。

◉

Voice / TTS Ready

輸出可接 Lu voice 或 Hermes TTS 的敘述腳本，後續可轉成補課影片或課堂旁白。

Multi-Agent Interaction

Dynamic classroom engagement modes

多代理不是裝飾，而是把真實課堂裡不同角色的認知任務拆開：講解、追問、挑錯、提出學生可能的反應。

Classroom Discussion

AI classmates 提出常見直覺答案，讓老師可以把迷思顯性化，而不是直接公布公式。

Roundtable Debate

不同代理用圖像、公式、實驗與反例辯論同一題，協助學生看見物理表徵之間的轉換。

Q&A Mode

學生可用口語問「為什麼不能這樣代？」助教回到條件、系統與方向約定。

Architecture

Technical foundation

Lesson generation pipeline 分兩階段運作。第一階段分析使用者輸入的物理主題、學生樣貌與課堂時間，產生 structured lesson outline。第二階段將每個 outline item 轉成 rich scenes：slides、quiz、simulation、PBL activity 或 whiteboard script。

若需要真正 LLM 生成，公開 GitHub Pages 只呼叫受控 proxy；proxy 再使用盧老師 WSL 內已登入的 Hermes OpenAI Codex OAuth。這保留 OpenMAIC 形式的生成體驗，也維持安全邊界。

Research

Academic foundation & local teaching adaptation

原始 OpenMAIC 概念來自 MAIC（Multi-Agent Interactive Classroom）研究脈絡：從 MOOC 的被動影片，轉向由 LLM agents 生成、討論、評量與互動的主動學習環境。

本頁不是聲稱複製完整 OpenMAIC 平台，而是將其 landing page 與教室概念轉譯成盧老師可展示、可迭代的繁中物理 prototype。

FAQ

Frequently asked questions

Is this the official OpenMAIC platform?

No. This is a local teaching prototype inspired by OpenMAIC and adapted for Traditional Chinese high-school physics. It clearly separates the original research concept from this classroom-design demo.

Does the public page store API keys?

No. The public page does not store API keys or OAuth tokens. When real model generation is needed, the browser calls a controlled Hermes proxy and the credential stays server-side.

Can it run real AI generation?

The public demo works with a deterministic template. If a local /codex-course proxy is running, the same input can be routed through Hermes Codex OAuth and returned as JSON.
