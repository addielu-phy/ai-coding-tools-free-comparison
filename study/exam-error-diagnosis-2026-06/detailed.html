自然／理化：浮力與密度逐題詳解

自然 Q1正答 A能力：浮力＝排開液體重量

同質量鋁塊與鐵塊完全沒入水中，誰受浮力較大？

同質量的鋁塊（密度 2.7 g/cm³）和鐵塊（密度 7.8 g/cm³）沒入水中，鐵塊沉入水面下 20 公分、鋁塊沉入水面下 10 公分，何者受浮力較大？

A 鋁塊同質量下，密度較小代表體積較大；完全浸入同一液體時，排開水體積較大，浮力較大。
B 鐵塊錯在把「密度大／較重的感覺」誤當成浮力大。浮力不直接看密度大，而看排開液體體積。
C 一樣大若兩者體積相同且完全浸入才會一樣大；本題同質量不同密度，體積不同。
D 無法比較可比較，因為密度與質量足以判斷體積大小，再判斷浮力。
錯誤樣態：把「物體重量／密度」當成浮力大小依據。
下次策略：先問：是否完全浸入？若是，浮力看排開體積。
自然 Q3正答 C能力：浮力與視重

鐵塊在水中秤得 28 公克重，空氣中重量是多少？

密度 8 g/cm³ 的鐵塊完全沉入水中，若在水中所測得重量為 28 公克重，求空氣中重量。

A 24太小，忽略了在水中測得的是「扣掉浮力後的視重」。
B 28若選此，代表把水中視重當成真實重量，忘了浮力會使秤重變小。
C 32設體積 V，空氣中重量 8V，浮力約 V，水中視重 8V−V=7V=28，所以 V=4，真重 32。
D 36高估，通常是把浮力補太多或比例算錯。
錯誤樣態：不熟「真重－浮力＝視重」。
下次策略：水中秤重題一定先寫：視重 = 物重 − 浮力。
自然 Q4正答 D能力：沉浮圖判讀

同體積方塊在水中位置不同，質量大小如何？

甲、乙、丙三種材質不同但體積相同的正立方體置入水中，依圖判斷三者質量大小。

A 甲＝乙＝丙位置不同代表密度不同；同體積下，密度不同則質量不同。
B 甲＝乙＞丙沉在下方的丙密度最大，不可能質量最小。
C 丙＞甲＝乙丙最大合理，但甲乙浸入比例不同，表示密度不同，質量不等。
D 丙＞甲＞乙同體積時，越沉代表密度與質量越大；丙沉底最大，甲比乙浸入多，所以甲＞乙。
錯誤樣態：只看「浮或沉」，沒有再看漂浮者浸入比例。
下次策略：同體積題：位置越低／浸入越多，密度與質量通常越大。
自然 Q5正答 B能力：同質量不同密度的體積比較

同質量鐵球與石頭放入水中，誰受浮力較大？

已知鐵的密度比石頭大，今將質量相同的實心鐵球與石頭放入水中，所受浮力如何？

A 鐵球較大錯在以為密度大就浮力大；同質量下鐵球密度大，體積反而小。
B 石頭較大同質量下，密度較小者體積較大；若皆浸入水中，排開水較多，浮力較大。
C 兩者相同只有體積相同且完全浸入時才相同；本題同質量不同密度，體積不同。
D 無法比較可比較，因為題目給了密度大小與質量相同。
錯誤樣態：混淆「密度大」與「體積大」。
下次策略：同質量比較體積：密度越大，體積越小。
自然 Q8正答 B能力：漂浮平衡

雞蛋從沉底到加鹽後浮出水面，哪個判斷錯？

雞蛋放入水中沉底，在水中加鹽後浮出水面。問下列判斷何者錯誤。

A 鹽越多，浮出水面體積越多通常正確。鹽水密度變大，雞蛋要排開較少體積就能平衡，所以露出較多。
B 雞蛋重量小於在鹽水中所受浮力錯。漂浮靜止時是力平衡，浮力等於重量，不是大於重量。
C 雞蛋重量大於在純水中所受浮力正確。純水中沉底，表示重力大於浮力，還需底部支持力。
D 雞蛋密度小於鹽水密度能浮在鹽水時，平均密度小於鹽水密度，正確。
錯誤樣態：以為「浮起來」代表浮力大於重力。
下次策略：靜止漂浮一定是浮力＝重力；只有加速上升瞬間才可能浮力＞重力。
自然 Q10正答 C能力：浮力－物重＝需施力

木塊完全壓入水中，要施力多少？

密度 0.6 g/cm³、體積 250 cm³ 的木塊完全壓入水中，手至少要施力多少公克重？

A 200高估，可能把浮力 250 減錯或物重算錯。
B 150這是木塊本身重量：0.6×250=150 gf，不是手要施的力。
C 100完全浸入時浮力 250 gf，木塊重量 150 gf；要向下壓住，需施力 250−150=100 gf。
D 50低估，通常是把差值算錯或漏掉完全浸入時排開 250 cm³ 水。
錯誤樣態：把「木塊重量」和「手施力」混在一起。
下次策略：畫受力圖：上浮力 250，下重力 150，下手力 F；平衡：250=150+F。

數學：平行線與角度題逐題講解方向

數學多題共通需補答案卷可逐題精修

平行線角度題的選項判斷法

照片中數學卷多題涉及平行線、角平分線、外角、反射路徑與輔助線。部分題幹與紅筆正答可辨識，但細節不足以保證逐題正答完全無誤，因此先整理「每類選項錯法」。

選項若使用同位角／內錯角相等只有在確認兩線平行，且角型為 F 型或 Z 型時才成立。這類通常是正解核心。
選項若把同側內角當相等常是錯的。同側內角多數是互補，應相加 180°。
選項若漏掉角平分線 ÷2角平分線題常錯在沒有把大角分成兩半，或把兩段角誤認為原角。
選項若不畫輔助平行線折線、轉彎、球檯反射題常需過轉折點畫平行線，否則會把角加錯。
錯誤樣態：看到圖就猜，沒有在圖上標「平行線、截線、Z/F/U 型」。
下次策略：每題先寫三句：哪兩線平行？哪條是截線？這題是 Z、F 還是 U？
數學計算題能力：解題書寫

計算題扣分的主要原因

照片中的計算題可見有紅筆扣分，代表不是只有答案問題，也包含過程不完整或角度追蹤錯。

A 只寫答案即使答案接近，若沒有標明角度來源，容易被扣過程分。
B 輔助線沒說明畫輔助線後要寫「過某點作平行線」，否則老師看不出依據。
C 角度關係沒註明例如「內錯角相等」「同側內角互補」「三角形內角和 180°」要寫出來。
D 完整做法圖上標角 → 寫角度關係 → 列式 → 算答案 → 回填檢查。
錯誤樣態：會算一部分，但缺乏可得分的推理文字。
下次策略：每題至少寫一個理由詞：內錯角、同位角、同側內角、外角、角平分線。

國文：〈空城計〉錯題選項分析方向

國文閱讀題照片文字較密，先依可辨識錯法整理

〈空城計〉選項常見錯誤樣態

國文卷錯題集中在字詞、成語、人物心理與閱讀題。由於照片中文字密集，部分選項無法百分之百精確 OCR，以下先做「選項類型」詳解。

選項有文本依據例如能連到諸葛亮「沉著、機智、善用心理戰」或司馬懿「多疑、謹慎」的敘述，通常較合理。
選項把人物心態說反如把諸葛亮說成慌亂、把司馬懿說成魯莽進攻，通常不符文本。
選項過度推論文本只說某人疑懼退兵，選項卻延伸成完全知道真相，屬於說太滿。
選項只符合片段字面看似抓到某個詞，但沒有符合整段主旨或人物目的。
錯誤樣態：被「半對半錯」選項吸引。
下次策略：每個選項標 S/I/X/O：文本支持、合理推論、無證據、說太滿。

公民：行政法規與救濟逐選項分析

公民核心錯題群能力：三種責任分類

民事、刑事、行政責任的選項判斷

公民卷多題在問「這是哪一種責任／救濟／法規」。這類題常因生活案例相似而選錯。

A 民事責任人民與人民之間的損害賠償、契約、侵權，例如弄壞手機要賠錢。
B 刑事責任犯罪行為，由國家追訴，可能有罰金、拘役、有期徒刑。
C 行政責任違反行政法規，由行政機關處罰，例如罰鍰、吊照、停業、限期改善。
D 政治責任多與民選公職、政治承擔有關，國中行政法規題較少以此作一般違規答案。
錯誤樣態：看到「罰錢」就以為是刑事罰金或民事賠償。
下次策略：先問誰處罰？若是行政機關依行政法規開罰，多半是行政責任。
公民救濟流程正確流程

不服行政處分：選項應怎麼判斷？

人民不服行政機關處分時，常見選項會混入民事訴訟、刑事訴訟、訴願、行政訴訟。

A 民事訴訟錯。民事訴訟處理私人間權利義務，不是挑戰行政處分的第一步。
B 刑事訴訟錯。刑事訴訟處理犯罪追訴，不是對行政機關處分不服的救濟。
C 訴願通常正確第一步。對行政處分不服，先向上級或原機關提出訴願。
D 行政訴訟不是第一步；通常是訴願結果仍不服後，才進入行政法院。
錯誤樣態：把所有「上法院」都當成第一步。
下次策略：背流程：行政處分 → 訴願 → 行政訴訟。
公民法規位階能力：法律保留／命令不得牴觸法律

法律、命令、自治條例的選項判斷

題目若問行政機關或地方政府訂規範，需判斷是否符合上位法。

A 憲法／法律位階較高法律、命令、自治條例都不能牴觸上位法。
B 行政命令可任意限制人民權利錯。限制人民權利通常需要法律授權或法律保留。
C 地方自治條例可違反中央法律錯。地方自治也受法律與憲法限制。
D 機關內部規則可直接處罰人民錯。對人民不利的處分或罰則需有法規依據。
錯誤樣態：以為「政府說了就算」。
下次策略：先檢查：有沒有法律依據？有沒有違反上位法？
