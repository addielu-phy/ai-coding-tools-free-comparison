自然／理化：浮力與密度——孩子選錯選項分析

自然 Q1孩子疑似選 C：一樣大正答 A：鋁塊

同質量鋁塊與鐵塊完全沒入水中，誰受浮力較大？

同質量的鋁塊（密度 2.7 g/cm³）和鐵塊（密度 7.8 g/cm³）沒入水中，何者所受浮力較大？

孩子答案疑似 C：一樣大
正確答案A：鋁塊
核心誤解以為同質量 → 浮力一樣，忽略體積不同。
A 鋁塊正確。同質量下，密度較小的鋁體積較大；完全浸入同一液體時，排開水體積較大，浮力較大。
B 鐵塊錯。這是把密度大、感覺較重誤當浮力大。
C 一樣大孩子若選這個，表示只抓到「同質量」，但沒有轉成「同質量 + 不同密度 → 不同體積」。同質量不代表浮力相同。
D 無法比較錯。題目已給質量相同與密度大小，足以比較體積與浮力。
錯誤樣態：把「同質量」直接推成「浮力相同」。
補救句：浮力比較先看排開液體體積；同質量時，密度越小，體積越大。
自然 Q3孩子選 B：28正答 C：32

鐵塊在水中秤得 28 公克重，空氣中重量是多少？

密度 8 g/cm³ 的鐵塊完全沉入水中，若在水中所測得重量為 28 公克重，求空氣中重量。

孩子答案B：28
正確答案C：32
核心誤解把水中視重當成真實重量。
A 24錯，方向更小，代表可能把浮力又扣了一次。
B 28孩子選這個，表示沒有意識到「水中秤重會變小」。28 是視重，不是空氣中真重。
C 32正確。設體積 V，真重 8V，浮力 V，視重 8V−V=7V=28，所以 V=4，真重 32。
D 36錯，補回浮力過多，比例計算錯。
錯誤樣態：沒有建立「真重 − 浮力 = 視重」。
補救句：只要是在液體中用彈簧秤量到的重量，都要先想：它已經被浮力抵掉一部分。
自然 Q4孩子作答痕跡不清依紅筆疑似正答 D

同體積方塊在水中位置不同，質量大小如何？

甲、乙、丙體積相同，置入水中後位置不同，判斷三者質量大小。

孩子答案照片中不夠清楚
正確答案疑似 D：丙＞甲＞乙
可能誤解沒有用浸入比例判斷密度。
A 甲＝乙＝丙錯。位置不同代表密度不同；同體積下質量也不同。
B 甲＝乙＞丙錯。丙沉得最明顯，不可能質量最小。
C 丙＞甲＝乙部分合理但不完整；甲乙浸入比例不同，質量不應相等。
D 丙＞甲＞乙正確方向。同體積時，越沉代表密度與質量越大。
錯誤樣態：只看「浮／沉」，沒有比較漂浮物浸入水中的比例。
補救句：同體積題：誰浸得越深，誰密度越大、質量越大。
自然 Q5孩子選 C：兩者相同正答 B：石頭較大

同質量鐵球與石頭放入水中，誰受浮力較大？

鐵的密度比石頭大，質量相同的實心鐵球與石頭放入水中，所受浮力如何？

孩子答案C：兩者相同
正確答案B：石頭較大
核心誤解以為同質量或都沉入水中 → 浮力相同。
A 鐵球較大錯。密度大不代表浮力大；同質量下鐵球體積較小。
B 石頭較大正確。石頭密度較小，同質量時體積較大，排開水較多，浮力較大。
C 兩者相同孩子選這個，表示把「質量相同」誤認為浮力相同，沒有用密度換算體積。
D 無法比較錯。題目給了密度大小與質量相同，足以比較。
錯誤樣態：質量、密度、體積三者轉換不熟。
補救句：同質量比較體積：密度越大，體積越小；完全浸入時體積越小，浮力越小。
自然 Q8孩子作答疑似不是 B正答 B：浮力大於重量是錯的

雞蛋從沉底到加鹽後浮出水面，哪個判斷錯？

雞蛋在純水中沉底，加鹽後浮出水面，判斷何者錯誤。

孩子答案照片中圈選不夠清楚，疑似未選 B
正確答案B
核心誤解不清楚「漂浮靜止」時浮力與重量相等。
A 加鹽越多，露出水面體積越多通常正確。鹽水密度變大，排開較少體積即可平衡。
B 雞蛋重量小於在鹽水中所受浮力這是錯誤敘述。漂浮靜止時浮力＝重量，不是浮力大於重量。
C 雞蛋重量大於在純水中所受浮力正確。沉底時，重力大於浮力，還有底部支持力。
D 雞蛋密度小於鹽水密度正確。能浮起表示雞蛋平均密度小於鹽水密度。
錯誤樣態：把「浮起來」誤解成「浮力一直大於重量」。
補救句：靜止漂浮：浮力＝重量；正在加速上升時才可能浮力＞重量。
自然 Q10孩子選 D：50正答 C：100

木塊完全壓入水中，要施力多少？

密度 0.6 g/cm³、體積 250 cm³ 的木塊完全壓入水中，手至少要施力多少？

孩子答案D：50
正確答案C：100
核心誤解受力平衡式沒有列完整，或把重量、浮力、手力差值算錯。
A 200錯，可能把浮力與重量關係反過來或高估手力。
B 150這是木塊重量：0.6×250=150 gf，不是手施力。
C 100正確。完全浸入浮力 250 gf，木塊重量 150 gf；需向下施力 F，使 250=150+F，所以 F=100。
D 50孩子選這個，表示可能只做了部分差值，或沒有把「完全浸入時排開 250 cm³ 水」轉成 250 gf 浮力。
錯誤樣態：沒有畫受力圖，只憑直覺找差。
補救句：壓住漂浮物：上方浮力 = 下方重量 + 手向下的力。

數學：已辨識的孩子作答與錯誤樣態

數學卷多題可見孩子選項，但部分正答需答案卷確認

選擇題與填充題的錯誤樣態

可見孩子在多題圈選 B、C、D，紅筆多處打叉。由於部分紅筆正答與題幹被折痕遮住，這裡先把「孩子為什麼會選到這類錯選項」整理出來。

孩子常選：看起來角度最接近的選項可能沒有逐步標角，而是看圖估角或從選項猜。
孩子常錯：同側內角與內錯角混淆若把 U 型同側內角當成相等，就會選到差一個 180° 互補關係的答案。
孩子常錯：忘記畫輔助平行線折線、球檯反射、轉彎角題若不補平行線，很容易把角直接相加或相減錯。
正確解題流程先標平行線與截線 → 判斷 Z/F/U 型 → 寫角度關係 → 列式 → 檢查是否需角平分線或外角。
錯誤樣態：不是完全不會角度，而是「圖形標記流程」不穩。
補救句：看到平行線題，不准先算；先在圖上標 Z、F、U。

〈空城計〉閱讀與字詞題錯誤樣態

可見國文卷多題有圈選與紅筆更正，但題幹與選項較密集。先依可見錯題型態分析孩子為何會選錯。

選到「半對半錯」選項表示孩子抓到部分字面，但沒有回到全文脈絡確認。
把人物心理說反〈空城計〉的重點是諸葛亮沉著用計、司馬懿多疑退兵；若選項把人物心態反過來，就會錯。
字詞義不熟造成連鎖錯注音、解釋、成語若不穩，閱讀題會更容易被選項牽著走。
正確檢查方式每個選項標 S/I/X/O：文本支持、合理推論、無證據、說太滿。
錯誤樣態：閱讀題不是看不懂全文，而是選項辨識不精準。
補救句：選項只要有一半超出文本，就不能選。

公民：孩子選錯選項背後的觀念混淆

公民卷可見多題圈選與紅筆更正

行政責任、行政救濟與法規位階

公民卷多題屬於生活案例判斷，孩子容易看到「罰錢、法院、政府」就選錯類別。

把行政罰當刑事責任若看到罰錢就想到犯罪，會把罰鍰與罰金混淆。行政機關開的是罰鍰，法院刑事判決才常見罰金。
把行政救濟直接跳到訴訟不服行政處分通常先訴願，再行政訴訟；若直接選法院，流程會錯。
以為政府命令可以任意限制人民行政命令與自治規則都不能牴觸法律，限制人民權利通常要有法律依據。
正確判斷流程先問：誰對誰？是私人糾紛、犯罪，還是行政機關依法處分？再決定民事／刑事／行政。
錯誤樣態：生活案例分類能力不足，不是單純背名詞不足。
補救句：公民題先分類再選答案：人民對人民＝民事；犯罪＝刑事；違反行政規定或機關處分＝行政。
