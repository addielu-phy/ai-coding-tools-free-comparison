A. 自然／理化：浮力與密度

看到的錯題型態

這張自然卷幾乎整份都圍繞浮力與密度，學生在以下題型常錯：

  1. 比較不同物體在水中或液體中的浮力大小。
  2. 判斷物體會浮、沉、懸浮。
  3. 由密度與體積求重量或排水量。
  4. 看圖判斷甲、乙、丙誰受浮力較大。
  5. 判斷加入鹽水後浮沉變化。
  6. 混淆「物體重量」、「浮力」、「排開液體重量」、「密度」。

核心錯因

1. 沒有把浮力核心公式背成一個判斷句

應建立：

浮力 = 排開液體的重量。
同一液體中，排開體積越大，浮力越大。
完全沉入時，浮力看體積與液體密度，不直接看物體本身重量。

很多錯題看起來是把「重的物體浮力一定大」當成直覺，但這不一定對。

2. 沉浮條件不熟

要熟：

  • 物體密度 < 液體密度：浮起
  • 物體密度 = 液體密度：懸浮
  • 物體密度 > 液體密度：下沉

以及：

  • 漂浮時：浮力 = 物重
  • 完全沉入且靜止懸浮時：浮力 = 物重
  • 沉在底部時：浮力 < 物重，底部還有支持力

3. 題目問「最大浮力」時，沒有先判斷是否完全浸沒

若物體完全浸入同一液體：

體積越大，浮力越大。

若物體漂浮：

浮力等於物體重量。

這兩種情況常常混在一起。

補強練習

自然練習 1：一句話判斷

請快速回答：

  1. 同一杯水中，兩個體積相同、密度不同的鐵塊和鋁塊完全沉入水中，浮力誰比較大？
  2. 同一杯水中，體積大的木塊和體積小的木塊都漂浮，浮力誰比較大？
  3. 一顆球從清水移到鹽水，若都漂浮，浮力變大、變小、還是不變？
  4. 一顆球從清水移到鹽水，露出水面的部分變多還是變少？
  5. 一個物體密度 0.8 g/cm³，水密度 1.0 g/cm³，它會浮還是沉？

自然練習 2：計算題

  1. 某物體體積 100 cm³，完全浸入水中，水密度 1 g/cm³，排開水重約多少 gf？
  2. 某物體密度 0.7 g/cm³，體積 100 cm³，漂浮於水面，物體重量約多少 gf？浮力約多少 gf？
  3. 某木塊完全壓入水中時受浮力 250 gf，若手要讓它完全壓入水中，至少要施力 100 gf，木塊本身重量是多少 gf？
  4. 一個 1.5 kgw 的無蓋鐵盒漂浮，最多可再放多少重量而不沉，取決於什麼？

自然練習答案

  1. 體積相同、同液體、完全浸入，浮力一樣大。
  2. 都漂浮時，浮力等於物重；若體積大木塊也比較重，則浮力較大，但要看重量。
  3. 若都漂浮，浮力等於物重，所以浮力不變。
  4. 鹽水密度較大，需排開較少體積即可平衡，所以露出較多。
  5. 密度小於水，會浮。
  6. 100 gf。
  7. 物重 70 gf，漂浮時浮力 70 gf。
  8. 完全浸入時浮力 250 gf，手向下 100 gf，表示物重約 150 gf。
  9. 取決於鐵盒可排開的最大水重扣掉鐵盒本身重量。

B. 數學：平行線與角度追蹤

看到的錯題型態

數學卷主題很集中：

  • 平行線截角
  • 同位角、內錯角、同側內角
  • 三角形內角和
  • 外角定理
  • 角平分線
  • 折線或球檯反射題
  • 需要畫輔助線的題目

分數約 63，表示不是完全不會，而是遇到複雜圖形會失控。

核心錯因

1. 圖形標記習慣不足

很多角度題不能只看選項猜，要做三步：

  1. 先把已知角度寫在圖上。
  2. 把平行線產生的同位角／內錯角標出來。
  3. 再用三角形內角和或外角定理追角。

2. 看到平行線時，沒有主動找「Z、F、U」

記法：

  • Z 型：內錯角相等
  • F 型：同位角相等
  • U 型：同側內角互補

3. 多步驟角度題缺少檢查

常見錯誤：

  • 把互補 180 度誤當相等
  • 把外角當內角
  • 忘記三角形內角和 180 度
  • 有角平分線卻沒有除以 2

補強練習

數學練習 1：看到平行線先問三件事

每題都要寫：

  1. 哪兩條線平行？
  2. 哪條線是截線？
  3. 出現的是 Z 型、F 型還是 U 型？

數學練習 2：角度追蹤

  1. 兩平行線被一直線截，若一個角是 65°，它的同位角是多少？同側內角是多少？
  2. 三角形兩角為 45°、65°，第三角是多少？
  3. 三角形一外角為 120°，其中一個不相鄰內角為 50°，另一個不相鄰內角是多少？
  4. 若角平分線把一個 80° 的角分成兩等分，每一個是多少？
  5. 兩平行線被折線連接，若上方夾角 35°，下方夾角 42°，中間轉角通常要把輔助線畫成什麼？

數學練習答案

  1. 同位角 65°；同側內角 115°。
  2. 70°。
  3. 70°。
  4. 40°。
  5. 畫一條通過折點且與兩條平行線平行的輔助線，再分別使用內錯角／同位角。

C. 國文：〈空城計〉與閱讀理解

看到的錯題型態

國文卷可見：

  • 注音、字形、字義扣分
  • 詞語解釋扣分
  • 改錯題扣分
  • 成語活用題有錯
  • 〈空城計〉人物心態與文意理解題錯
  • 閱讀測驗題組有錯

核心錯因

1. 字詞基本功不穩

注音、字義、詞語解釋若先扣掉一大段，後面閱讀就更難穩住。

2. 文意題沒有抓「人物目的」

〈空城計〉的核心不是只記情節，而是要抓：

  • 諸葛亮為何敢開城門？
  • 司馬懿為何不敢進攻？
  • 兩人各自判斷的依據是什麼？
  • 作者如何塑造人物智慧與心理攻防？

3. 閱讀測驗選項辨析不足

國文閱讀題常見陷阱：

  • 選項有一半對，但範圍過大
  • 選項是文本沒有說的推論
  • 選項把人物態度講反
  • 選項使用絕對語氣，例如一定、完全、都

補強練習

國文練習 1：空城計三問

  1. 諸葛亮使用空城計的真正目的是什麼？
  2. 司馬懿退兵的心理原因是什麼？
  3. 這篇文章塑造諸葛亮哪一種形象？

國文練習 2：閱讀選項判斷

看到選項時標記：

  • S：文本有直接支持
  • I：可合理推論
  • X：文本沒有證據
  • O：說太滿或過度推論

國文練習答案方向

  1. 以虛張聲勢、反用對方多疑心理，拖延或避免敵軍進攻。
  2. 司馬懿熟知諸葛亮謹慎，認為城中必有埋伏，因此退兵。
  3. 沉著、機智、善用心理戰、臨危不亂。

D. 公民：行政法規與行政救濟

看到的錯題型態

公民卷主題是：

  • 行政責任
  • 行政罰
  • 行政處分
  • 行政命令
  • 行政救濟
  • 訴願、行政訴訟
  • 法律、命令、自治條例
  • 民事責任、刑事責任、行政責任區分

分數約 54，代表概念混淆不少。

核心錯因

1. 三種責任混在一起

最重要先分清楚：

2. 行政救濟流程不熟

基本流程：

不服行政處分 → 訴願 → 行政訴訟

常見錯誤是把「訴願」和「行政訴訟」順序搞錯，或把法院／行政機關角色混淆。

3. 法規位階混淆

要先記：

憲法 > 法律 > 命令 > 自治條例／自治規則

行政命令不能違反法律，法律不能違反憲法。

4. 看到生活案例時，沒有先判斷「誰對誰」

公民題要先問：

  1. 是人民對人民？
  2. 是人民違反國家行政規定？
  3. 是國家機關作成處分？
  4. 是不服處分要救濟？

這四個一分清楚，答案會穩很多。

補強練習

公民練習 1：責任分類

請判斷是民事、刑事還是行政責任：

  1. 騎車違規停車，被開罰單。
  2. 打破同學手機，需要賠償。
  3. 酒駕造成他人受傷，被法院判刑。
  4. 餐廳違反衛生規定，被勒令停業。
  5. 亂丟垃圾，被環保局罰鍰。

公民練習 2：行政救濟流程

  1. 人民不服戶政事務所拒絕改名，應先向哪裡尋求救濟？
  2. 若訴願仍不服，下一步是什麼？
  3. 行政訴訟是向普通法院還是行政法院？
  4. 行政機關發布的命令可以違反法律嗎？
  5. 地方政府制定自治條例，可以違反中央法律嗎？

公民練習答案

  1. 行政責任。
  2. 民事責任。
  3. 刑事責任，也可能另有民事賠償。
  4. 行政責任。
  5. 行政責任。
  6. 先訴願。
  7. 行政訴訟。
  8. 行政法院。
  9. 不可以。
  10. 不可以。
