國二學習資源

把目前已公開的國中相關網頁整理成學生與老師可直接使用的入口；版面先依「科目」分類，再在每個科目底下細分「錯題診斷、學習重點、互動題庫、自學評量、教師端、會考複習、翻卡練習、舊入口」等網頁類型。

建議使用順序

  1. 先看「錯題診斷」找弱點。
  2. 理化用第 5、6 章與浮力／摩擦力題庫練核心觀念。
  3. 老師可從「教師端」查看國二下理化期末自學評量全班紀錄。
  4. 國文先做期末注音國字／注釋自學評量，再用課文翻卡每天短練。

整理原則

  • 先依科目分類，再在科目底下依網頁類型分組，讓學生／老師先找科目再找用途。
  • 老師介面會放在對應科目下的「教師端」類型，並標示「教師端需登入」。
  • 舊入口保留在「備用入口」區，避免歷史連結失效，但優先引導到新版頁。
  • 另 repo 的公開題庫也放入，讓學生不用到總覽頁逐一搜尋。

  • 補上國二上、國三會考複習與各章節學習地圖。
